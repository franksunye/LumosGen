**测试目标：** 验证LumosGen VS Code扩展的所有核心功能，确保用户体验流畅，系统稳定可靠

**测试范围：** 
- 核心功能测试
- 集成测试
- 用户体验测试
- 性能测试
- 错误处理测试
- 配置测试

**测试环境要求：**
- VS Code 1.74.0+
- Node.js 16+
